# 📊 DataHire: Salary Prediction & Analysis using Python and SQL

## 📌 Project Overview

DataHire is a data analysis project focused on exploring and understanding salary trends across different job roles, experience levels, education backgrounds, and industries.

The goal of this project is to use **Python (Pandas)** and **MySQL** to clean, process, and analyze a job salary dataset to extract meaningful insights.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* MySQL
* SQLAlchemy
* PyMySQL

---

## 🔄 Workflow

1. Loaded CSV dataset using Pandas
2. Cleaned and prepared data
3. Stored data into MySQL database
4. Performed analysis using SQL queries
5. Extracted insights from the dataset

---

## 📊 Key Analysis Performed

* Average salary overall and by job title
* Top 5 highest-paying job roles
* Salary trends based on experience
* Salary comparison across education levels
* Impact of skills and certifications on salary
* Industry-wise salary analysis
* Remote vs on-site salary comparison

---

## 💡 Key Insights

* Salary increases significantly with experience
* Certain job roles offer consistently higher salaries
* Higher skill count is associated with better pay
* Certifications can positively impact salary
* Some industries and locations offer higher compensation
* Remote jobs provide competitive salary opportunities

## ▶️ How to Run the Project

1. Clone the repository
2. Install required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Create a MySQL database (e.g., `datahire`)
4. Update database credentials in `data_loading.py`
5. Run the script:

   ```
   python src/data_loading.py
   ```
6. Execute SQL queries from `analysis_queries.sql` in MySQL
